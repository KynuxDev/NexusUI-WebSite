## 🚀 Kurulum ve Çalıştırma

```bash
# Repoyu klonlayın
git clone https://github.com/KynuxDev/NexusUI-WebSite.git

# Proje klasörüne gidin
cd NexusUI-WebSite

# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev
```

<div align="center">
  <p>🔄 Tarayıcınızda otomatik olarak <code>http://localhost:5173</code> açılacaktır</p>
</div>

## 🎨 Özelleştirme

Siteyi kendi botunuza uyarlamak için aşağıdaki değişiklikleri yapabilirsiniz:

<details>
<summary><b>🤖 Bot Bilgilerini Değiştirme</b></summary>

`src/components/Hero.tsx` ve diğer bileşenlerde bot bilgilerini güncelleyebilirsiniz:

```tsx
// Hero.tsx içinde
<motion.h1>
  <span>YeniBotAdı</span>
  <span>Kendi Açıklamanız</span>
</motion.h1>
```
</details>

<details>
<summary><b>🎭 Renk Şemasını Değiştirme</b></summary>

Ana renkleri `tailwind.config.js` dosyasında güncelleyin:

```js
module.exports = {
  theme: {
    extend: {
      colors: {
        // Kendi renk şemanızı buraya ekleyin
        primary: {
          DEFAULT: '#7B2CBF', // Ana renk
          dark: '#3A0CA3',    // İkincil renk
        }
      }
    }
  }
}
```
</details>

<details>
<summary><b>📋 Komut Listesini Güncelleme</b></summary>

`src/components/Commands.tsx` dosyasındaki komut listesini kendi botunuza uygun şekilde güncelleyin:

```tsx
const commands = [
  { category: 'Kendi Kategoriniz', name: '/komut1', description: 'Komut açıklaması' },
  // Diğer komutlar...
];
```
</details>
